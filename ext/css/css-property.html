CSS Properties: The Styling Toolkit

CSS properties are the actual style rules you apply to selected elements. There are hundreds of properties available, each controlling different aspects of an element's appearance.

Typography Properties

Control how text appears on your page with these essential typography properties:

CSS
.article-text { font-family: 'Inter', sans-serif; font-size: 18px; font-weight: 400; line-height: 1.7; letter-spacing: 0.02em; color: #374151; text-align: left; }

Box Model Properties

The CSS box model is fundamental to understanding layout. Every element is a box with content, padding, border, and margin:

CSS
.card { /* Content size */ width: 300px; height: 200px; /* Spacing inside the element */ padding: 20px; /* Border around the element */ border: 2px solid #e5e7eb; border-radius: 12px; /* Spacing outside the element */ margin: 20px auto; }
