CSS (Cascading Style Sheets) is the language that brings life and personality to web pages. While HTML provides the structure, CSS adds the colors, layouts, typography, and visual polish that make websites beautiful and engaging. Understanding CSS fundamentals is essential for anyone looking to create compelling web experiences.

CSS Selectors: Targeting Elements

Selectors are the foundation of CSS. They determine which HTML elements your styles will affect. Think of selectors as the addressing system for your stylesâ€”they tell the browser exactly which elements to decorate.

Element Selectors

The most basic selector targets elements by their tag name. This applies styles to all instances of that element throughout your page.

CSS
p { color: #333; font-size: 16px; line-height: 1.6; } h1 { font-size: 2.5rem; font-weight: bold; }

Class Selectors

Class selectors are one of the most commonly used selectors. They target elements with a specific class attribute and are prefixed with a dot (.).

CSS
.button { background-color: #6366f1; color: white; padding: 12px 24px; border-radius: 8px; } .highlight { background-color: yellow; font-weight: bold; }

ID Selectors

ID selectors target a unique element with a specific ID attribute. They're prefixed with a hash (#) and should only be used once per page.

CSS
#header { background-color: #1f2937; padding: 20px; } #main-content { max-width: 1200px; margin: 0 auto; }
Pro Tip

Use classes for styling and IDs for JavaScript functionality. Classes are reusable and more flexible, while IDs should be unique to a single element on the page.

Advanced Selectors

Selector Example Description
Descendant div p Selects all <p> inside <div>
Child div > p Selects direct children <p> of <div>
Adjacent Sibling h2 + p Selects <p> immediately after <h2>
Attribute [type="text"] Selects elements with specific attribute
Pseudo-class a:hover Selects elements in a specific state
CSS - Advanced Selectors
/* Descendant selector */ nav a { text-decoration: none; } /* Pseudo-classes */ button:hover { background-color: #4f46e5; } input:focus { border-color: #6366f1; outline: none; } /* Attribute selector */ input[type="email"] { border: 2px solid #e5e7eb; }
